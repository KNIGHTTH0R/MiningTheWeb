Mapping the U.S. by Property Value Instead of Land Area - Markese Lavell Wider

My visualization critique is of a great visualization created by Max Galka. Max Galka created a cartogram of the residential property value for every county in the United States (the contiguous 48 states).

Max Galka created this cartogram using Carto3F. I am particularly intrigued by this visualization because without any information about what you are looking at it looks like some type of wild science experiment but with a little information provided this map tells a great story. From this map you can tell that most of the residential property value is located in a few concentrated areas in the United states.

You can see the full map here, an animated GIF here , and the full story here.

Visualizing Free Wi-fi Hotspot locations in New York City - Markese Lavell Wider

Here is my awesome CartoDB visualization using information provided via the NYC OpenData website. In this visualization you are able to see where free Wi-fi Hotspots are located in the city of New York as well as where they are clustered. I chose to use the Dark matter basemap and purple markers outlined with white in order to make the locations pop.

Map Hacking the Sisyphyus Subway map - Richard Dunks

This is an example visualization critique of a great visualization created by the master of interactive visualization and design, Richard Dunks. In this latest incarnation, he has remade a visualization from the Citizens Budget Commission using CartoDB, Leaflet, D3.js, and jQuery. While the original visualization showed the data in an interactive format using Tableau Public, Richard created a custom JavaScript map and was able to better display the data, providing multiple data layers and better interactivity with the data.

You can see the full map here and a blogpost about the map here.
